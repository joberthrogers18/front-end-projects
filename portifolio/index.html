Dr. Down

Sistema criado na disciplina “Métodos de desenvolvimento de software”, onde tinha o objetivo de gerenciar a ala de pacientes com síndrome de down do Hospital Regional da Asa Norte do Distrito Federal. Nesse projeto foram usado os princípios de metodologias ágeis de software no desenvolvimento da aplicação além de ser desenvolvido usando as tecnologia Django, HTML, CSS, Botstrap, Docker, Travis e Code climate.
